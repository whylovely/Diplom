### 2. API обязательств
**Приоритет: Высокий** | **Время: 3-4 ч**

**Новые файлы:**
- `Entities/ObligationEntity.cs`
- `Controllers/ObligationsController.cs`
- `Shared/Obligations/ObligationDto.cs`

**Структура ObligationEntity:**
```csharp
public class ObligationEntity
{
    public Guid Id { get; set; }
    public Guid UserId { get; set; }
    public string Counterparty { get; set; }  // Контрагент
    public decimal Amount { get; set; }
    public string Currency { get; set; }
    public int Type { get; set; }  // 0 = Долг (я должен), 1 = Задолженность (мне должны)
    public DateTimeOffset CreatedAt { get; set; }
    public DateTimeOffset? DueDate { get; set; }  // Срок погашения
    public bool IsPaid { get; set; }
    public DateTimeOffset? PaidAt { get; set; }
    public string? Note { get; set; }
}
```

**Эндпоинты:**
- [ ] `GET /api/obligations` — список обязательств
- [ ] `POST /api/obligations` — создание
- [ ] `PUT /api/obligations/{id}` — обновление
- [ ] `DELETE /api/obligations/{id}` — удаление
- [ ] `POST /api/obligations/{id}/pay` — отметить как оплаченное

## Миграции

После изменения Entities:
```bash
cd Server
dotnet ef migrations add <MigrationName>
dotnet ef database update
```
