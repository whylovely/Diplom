# План доработки клиентской части

## Текущая структура
- **Фреймворк**: Avalonia UI
- **Архитектура**: MVVM (CommunityToolkit.Mvvm)
- **Views**: AccountsView, CategoriesView, ReportView, JournalView, NewTransactionView, диалоги
