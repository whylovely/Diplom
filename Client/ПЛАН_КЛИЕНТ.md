# План доработки клиентской части

## Текущая структура
- **Фреймворк**: Avalonia UI
- **Архитектура**: MVVM (CommunityToolkit.Mvvm)
- **Views**: AccountsView, CategoriesView, ReportView, JournalView, NewTransactionView, ObligationsView, SettingsView
- **Диалоги** (Views/DialogViews): AddAccountDialog, AddCategoryDialog, AddObligationDialog, ConfirmDialog, FirstRunDialog, InputDialog, MessageDialog
- **Services**: MockDS (IDataService), NotificationService (INotificationService), InputDialogService (IInputDialogService), CategoryDialogService (ICategoryDialogService), SettingsService
- **Models**: Account, Category, Entry, JournalRow, MessageLevel, Money, MonthlyTotalRow, Obligation, Transaction, UserSettings
