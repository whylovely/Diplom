# Описание дизайна клиента (Avalonia UI)

Этот документ описывает визуальный стиль, компоненты и дизайн-систему клиентского приложения, реализованного на Avalonia UI. Дизайн ориентирован на современный темный интерфейс (Dark Mode) с акцентными цветами, градиентами и эффектами матового стекла (Acrylic).

## 1. Общая концепция и стиль
- **Тематика**: Глубокая темная тема с сине-фиолетовыми акцентами.
- **Стиль**: Чистый, минималистичный, с закругленными углами (Soft UI) и использованием карточек для группировки контента.
- **Вдохновение**: macOS (док-панель навигации, эффекты размытия), современные финтех-приложения.
- **Базовая тема Avalonia**: `FluentTheme`.

## 2. Цветовая палитра (Колор-коды)

### Фон (Backgrounds)
- **Главный фон окна**: `#080C1A` (Глубокий темно-синий/почти черный)
- **Фон панелей и карточек**: `#B0111528` (Полупрозрачный синеватый), `#0D1120`, `#151A2E`
- **Фон таблиц (DataGrid)**: `#0A0E1E`
- **Линии и границы (Borders)**: `#222845`

### Текст (Typography & Foreground)
- **Основной текст**: `#D8D8D8` (Светло-серый, комфортный для глаз)
- **Заголовки**: `#FFFFFF` (Белый)
- **Подзаголовки**: `#E0E0E0`
- **Второстепенный текст (Dim)**: `#9E9E9E`

### Акцентные цвета
- **Основной акцент (Кнопки, выделение)**: `#1E88C8` (Яркий синий)
- **Кнопки наведения (Hover)**: `#1E88C820` (Полупрозрачный синий)
- **Градиент акцентных кнопок**: от `#29B6F6` (Светло-голубой) до `#AA00FF` (Фиолетовый)
- **Градиент акцентных кнопок (Hover)**: от `#4FC3F7` до `#CE93D8`

### Статусные цвета (Финансы)
- **Доход (Позитив)**: `#00E676` (Ярко-зеленый) — Класс `TextBlock.moneyGreen`
- **Расход (Негатив / Опасность)**: `#FF5252` / `#E04848` (Красный) — Класс `TextBlock.moneyRed`

## 3. Типографика
- **Основной шрифт**: `Inter`, фоллбэк на `Segoe UI, sans-serif`.
- **Классы текста**:
  - `heading`: Размер 24, Bold, `#FFFFFF`. Используется для главных заголовков страниц.
  - `subheading`: Размер 16, SemiBold, `#E0E0E0`. Для подзаголовков внутри карточек.
  - `dim`: Размер 12, `#9E9E9E`. Для вспомогательных подписей, дат, пояснений.
  - `cyan`: Акцентный текст (`#29B6F6`), Bold.

## 4. UI Компоненты

### Кнопки (Buttons)
1. **Стандартная кнопка**: Фон `#151A2E`, граница 1px `#222845`, скругление `8px`. При наведении: Фон `#1E88C8`, текст `#FFFFFF`.
2. **Экстра-акцентная (Accent)**: Класс `.accent`. Используется линейный градиент. Без границ, со скруглением, белый жирный текст. Используется для главных действий (например, "Добавить", "Сохранить").
3. **Опасное действие (Danger)**: Класс `.danger`. Темный фон `#1F0F1C` с красным текстом `#E04848`. При наведении фон становится ярко-красным `#FF5252`.
4. **Кнопки нижней панели (Dock Items)**: Круглые прозрачные кнопки (`.dockItem`, `.navDock`) со скруглением `24px`. Иконки меняют цвет и фон при наведении (`#10FFFFFF`) и активности (`#20000000`, иконка `#5C9FD6`).

### Карточки (Cards)
Оборачивают логические блоки (например, баланс счета, настройки).
- **Большая карточка (`Border.card`)**: Скругление `16px`, Padding `20px`, граница `#222845`.
- **Малая карточка (`Border.cardSmall`)**: Скругление `12px`, Padding `14px`.

### Поля ввода и элементы управления
Все элементы управления (`TextBox`, `ComboBox`, `NumericUpDown`, `Expander`) имеют:
- Единый фон `#0D1120`.
- Цвет текста `#D8D8D8`.
- Границы `#222845`.
- Скругление (CornerRadius) `8px`.
- Фокус полей ввода (Textbox) подсвечивается синей границей (`#1E88C8`).

### Таблицы (DataGrid)
- Прозрачный фон строк (`#0A0E1E`).
- Горизонтальные линии разделения (`#151A2E`).
- Заголовки столбцов стилизованы под общий темный фон с уменьшенным серым текстом (Размер `12`, `#888888`).
- При наведении на строку (Hover) появляется легкая синяя подсветка (`#1E88C815`).

## 5. Макет и навигация (Layout & Navigation)
- Приложение не имеет стандартного верхнего или бокового меню.
- **Главный контейнер**: `MainWindow` делится на две части:
  1. Рабочая область сверху (с отступами `24,20,24,10`).
  2. Плавающая нижняя навигационная панель (Dock).
- **Плавающий Dock**:
  - Находится снизу по центру (`VerticalAlignment="Bottom", HorizontalAlignment="Center"`).
  - Скругление углов (`32px`), имеет границу и отбрасывает тень/размытие с использованием `ExperimentalAcrylicBorder` (`TintColor="#0A0E1A"`, `TintOpacity=0.2`).
  - Центральная кнопка навигации ("Новая операция") выделяется ярким синим фоном и размером.

## 6. Иконки
- В проекте не используются растровые изображения для иконок.
- Иконки реализованы через векторные пути XAML (`Path Data="..."`), что обеспечивает идеальную четкость при любом масштабировании и позволяет динамически менять цвет иконки через свойство `Fill` в зависимости от состояния (активно/неактивно/наведение).
